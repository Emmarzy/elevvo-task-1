**Useful Jupyter Shortcuts:**
- `Ctrl+Enter`: Run cell
- `Shift+Enter`: Run cell and move to next
- `Ctrl+M, M`: Convert cell to Markdown
- `Ctrl+M, Y`: Convert cell to Code
